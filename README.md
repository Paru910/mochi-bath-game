# もち風呂キャンミニゲーム (Mochi Bath Avoidance Mini-Game)

 Vtuber「もち」を主人公とした、風呂キャン（お風呂に入りたくない衝動）を回避してお風呂に導くリズム要素のあるミニゲームです。

## 🎮 ゲーム概要
もちさんの「お風呂に入りたくない！」という気持ちに寄り添ったり、時にはお風呂を促したりして、制限時間内に右側のお風呂へ連れて行きましょう。

- **操作方法 (PC)**:
  - `J` キー：否定 (👎) - もちさんの自堕落な気持ちに同意したり、引き止めたりします。
  - `K` キー：肯定 (👍) - お風呂を促したり、ポジティブな提案に同意したりします。
- **操作方法 (スマホ)**:
  - 画面下のボタンをタップして操作します。

## 🚀 遊び方
1. ゲームを開始し、難易度（かんたん / ふつう / むずかしい）を選択します。
2. リズムに合わせて出現するセリフを確認し、正しいリアクション（👍/👎）を返します。
3. 制限時間内に右端の「お風呂」に到達すれば TRUE END です！
4. 左端の「ベッド」に戻ってしまうと...？

## 🛠 カスタマイズ
`config.js` を編集することで、以下の内容を自由に変更できます：
- セリフの内容 (`TEXT_BANK`)
- BPM（ゲームのテンポ）
- 制限時間や移動量
